🏎️ Luxury Car Sales Platform
A high-end web application designed for the luxury automotive market. This project focuses on a premium user experience and the implementation of modern software architecture to handle complex business logic, such as real-time negotiations and proposal management.

🌟 Key Features
Premium Inventory: A curated showcase of luxury vehicles with high-fidelity imagery and detailed specifications.

Advanced Negotiation System: Our flagship feature. A dedicated negotiation hub where buyers and sellers can interact through a structured timeline.

Dynamic Proposal Timeline: Tracks every offer and counter-offer, providing a clear history of the bargaining process.

Interactive Decision Tools: Real-time actions for each proposal: Accept, Decline, or Counter-Offer.

Secure Authentication: User-specific dashboards for managing active negotiations.

🛠️ Tech Stack
Framework: ASP.NET Core Razor Pages

Database: Microsoft SQL Server

ORM: Entity Framework Core (Code First approach)

Frontend: HTML5, CSS3 (Luxury Design System), JavaScript (ES6+)

Icons & Styling: FontAwesome 6 & Google Fonts

🚀 The Challenge: Negotiation Logic
The most complex part of this development was the Proposal Management System.
We moved away from traditional static forms to an interactive timeline. This required:

Precise state management to ensure only the "Latest" offer is actionable.

Asynchronous updates (AJAX/Fetch) for counter-offers to prevent page reloads during the negotiation flow.

A "Luxury" UI/UX that handles complex data (IDs, Prices, Dates) without overwhelming the user.
